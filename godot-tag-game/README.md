# Godot Tag Game Template

A diecut template for creating multiplayer tag games in Godot 4.x.

## What This Creates

This template generates a complete multiplayer tag game with:

- **Multiplayer support** - Up to 3 players with keyboard and gamepad controls
- **Tag mechanics** - Players chase and tag each other
- **Bot players** - AI-controlled players for single-player mode
- **Complete game structure** - Scenes, scripts, and utilities ready to extend

## Usage

```bash
diecut new godot-tag-game
```

You'll be prompted for:
- **Project name** - Name of your game project
- **Author** - Your name
- **Description** - Brief description of your game
- **Godot version** - Godot version (default: 4.3)

## What's Included

- `project.godot` - Godot project configuration with input mappings
- `main.tscn` - Main game scene
- `level.gd` - Level management and game logic
- `scenes/` - Player character scenes and scripts
- `utils/` - Helper scripts for camera, scoring, UI elements
- Game assets (sprites, icons)

## Getting Started After Generation

1. Open the generated `project.godot` in Godot Editor
2. Press F5 to run the game
3. Start customizing the gameplay, graphics, and mechanics!

## Customization Ideas

- Add new player abilities or power-ups
- Create multiple levels or arenas
- Implement different game modes (freeze tag, infection, etc.)
- Add obstacles and environmental hazards
- Enhance visuals with particles and effects

## Requirements

- Godot 4.3 or later
- Controllers (optional, for multiplayer)
